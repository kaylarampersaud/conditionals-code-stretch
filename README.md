## Conditionals Code Stretch

- create a folder called conditionals-stretch in your code-stretches folder
- create an index.html filein your conditionals-stretch folder
- create a JS file and connect it to your HTML file
- create a variable that stores the answer (any number that you want)
- prompt the user to guess a number
- tell the user if the number is too high, low, or exactly correct
- push it up to GitHub
